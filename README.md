# Chess Engine

A lightweight, terminal-based chess engine written from scratch in C, featuring a built-in AI with adjustable thinking time.

## Features
* **Pure C Implementation:** Designed for high performance and efficiency.
* **Terminal Interface:** Play chess directly from your command line.
* **Adjustable AI Difficulty:** Set the engine's thinking time before starting. The longer it thinks, the stronger its moves become.
* **Standard Coordinate Notation:** Uses a clean and precise input method for moving pieces (e.g., `a2a4`).
* **Move Validation:** Built-in command to check legal moves in any position.

## Getting Started
How to Play

Launch and compile the engine. <br>
Specify color (w for white, b for black). <br>
Set AI thinking time (for example 10 for 10 sec) <br>
You can make your move by typing starting square followed immediately by the destination type (for example a2a4) <br>
You can type moves to find legal moves.
